========
Overview
========

Jstropic是什么意思？准确的说应该拆开来这个单词，js代表的是javascript，tropic是回归线，热带的意思。
Jstropic是一套体积很小的框架，是面向数据库到浏览器的数据处理框架，秉持少就是多的思想来提供强大的能力。

特征
--------

- 体积小
- 简单易用
- 无需编译
- 支持热部署
- 依托Java强大的开源生态
- 稳定可靠的JVM提供保障
- 可自由扩展

安装使用
------------

环境准备

* JDK1.8+
* Tropic
